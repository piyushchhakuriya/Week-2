# HireHub (minimal)
This archive contains a minimal frontend and backend project for the HireHub job portal.

## Backend
- Folder: `backend`
- Run:
  ```bash
  cd backend
  npm install
  npm run dev
  ```
- MongoDB required (or change MONGO_URI in `.env`)

## Frontend
- Folder: `frontend`
- Run:
  ```bash
  cd frontend
  npm install
  npm run dev
  ```
- Open http://localhost:5173 (Vite default) after running.
